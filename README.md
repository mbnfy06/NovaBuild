# NovaBuild - Plataforma Digital Premium

## Visión del Proyecto
Este sitio web ha sido diseñado y desarrollado con el objetivo estratégico de **posicionar a NovaBuild como un referente de lujo y exclusividad** en el sector de la construcción y reformas integrales en Valencia.

Más que una simple página informativa, esta plataforma actúa como una **herramienta de conversión de alto rendimiento**, diseñada para atraer a clientes que buscan calidad superior y atención al detalle.

---

## Objetivos Estratégicos

### 1. Imagen de Marca Premium
- **Estética "High-End"**: Uso de una paleta de colores oscura, tipografías modernas y espacios generosos para transmitir sofisticación.
- **Micro-interacciones**: Animaciones sutiles y efectos de paralaje que denotan cuidado y excelencia técnica, reflejando la calidad constructiva de la empresa.

### 2. Generación de Leads (CRO)
- **Call-to-Actions (CTA) Estratégicos**: Botones de contacto y WhatsApp siempre accesibles para maximizar las oportunidades de venta.
- **Formulario Optimizado**: Diseño limpio y accesible para reducir la fricción en el contacto.

### 3. Experiencia de Usuario (UX)
- **Navegación Intuitiva**: Estructura clara (Inicio -> Nosotros -> Proyectos -> Servicios -> Contacto) para guiar al usuario a través del embudo de ventas.
- **Rendimiento Ultrarápido**: Construido con tecnologías nativas (HTML5, CSS3, JS) sin dependencias pesadas, garantizando tiempos de carga mínimos y excelente puntuación en Google (SEO).

### 4. Portafolio Interactivo
- **Showcase de Proyectos**: Una sección dinámica que permite visualizar la calidad de los trabajos realizados, esencial para generar confianza en el cliente final.

---

## Especificaciones Ténicas
- **Stack Tecnológico**: HTML5 Semántico, CSS3 Moderno (Variables, Flexbox, Grid), JavaScript Vanilla (ES6+).
- **Rendimiento**: Optimizado para Core Web Vitals.
- **Responsive**: Diseño 100% adaptable a todos los dispositivos (Móvil, Tablet, Desktop).
- **Infraestructura**: Despliegue optimizado en Vercel con cabeceras de seguridad y caché.
